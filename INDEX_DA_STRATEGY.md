## 📈 QUALITY FRAMEWORK

### Scoring Methodology:
```
Per Image Evaluation:

Qualité visuelle        /10
Cohérence marque        /10
Composition             /10
Lumière & color grading /10
Modernité cinématique   /10
Crédibilité premium     /10
───────────────────────────
TOTAL                   /60

Percentage: Score ÷ 60 × 100
Acceptance: ≥ 95% (≥ 57/60)
```

### Quality Gates (Every Phase):
- ☑ Score ≥ 95/100 (absolute minimum)
- ☑ No watermarks or signatures
- ☑ Resolution meets spec exactly
- ☑ File size optimized
- ☑ Color accuracy verified
- ☑ Contrast ratio WCAG AA+
- ☑ No clichés detected
- ☑ Cinematic quality evident
- ☑ Narrative alignment confirmed
- ☑ Responsive display tested
- ☑ Performance impact minimal
- ☑ DA final approval signature

**Philosophy:** If image doesn't pass gates, regenerate it. Zero compromises.

## 💡 KEY PRINCIPLES

### Artistic Direction Philosophy:
1. **Zero Generics** - Every image intentional
2. **Narrative Cohesion** - Tell Studio's story
3. **Cinematic Mastery** - Professional production quality
4. **Brand Credibility** - Premium positioning evident

### Color Grading Strategy:
- Unified LUT across all images
- Shadows: Slightly theme-tinted
- Highlights: Warm white or theme-tinted

- Contrast: +15% (cinematic)
